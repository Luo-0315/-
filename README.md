1.项目名称与简介：Luo-0315的2026世界杯数据集原生AI比分预测平台，2026世界杯AI比分预测平台是一款基于真实世界杯历史赛事大数据与多机器学习模型集成的智能赛事预测可视化系统，基于Python+Streamlit开发，依托1930-2022年完整清洗结构化赛事数据集，实现科学化、数据化、可视化的世界杯对战比分与胜负概率智能推演。
平台整合球队场均进球、失球、红牌纪律、东道主优势、淘汰赛属性、现场观众等多维度赛事特征，集成随机森林、逻辑回归、支持向量机、决策树四大经典机器学习模型，支持自定义模型权重、赛制类型、东道主属性等进阶参数，通过多模型加权融合算法，大幅提升预测结果的客观性与精准度，规避传统主观预判的片面性问题。
系统具备全方位数据可视化能力，可实时生成球队六维能力评分对比、历年胜率趋势图谱、多模型性能评估图表，同时自动匹配球队历史交锋记录，并结合数据集统计数据生成专属预测研判依据。平台搭载动态UI动效、渐变视觉体系与交互卡片，界面美观流畅，兼具实用性与观赏性。同时内置5折交叉验证、混淆矩阵、多维度模型指标评测模块，可直观展示各算法准确率、F1分数、泛化稳定性，实现预测应用+算法评测+数据分析三位一体功能。
本项目摒弃简单公式推演，完全依托真实赛事数据训练建模，是一套轻量化、高适配、可拓展的体育AI数据分析与智能预测工具，可用于课程设计、科创项目、体育数据研究及赛事趣味推演。
2.功能说明：- 双球队对战AI预测：支持全球84支世界杯参赛球队自由配对，自动计算双方取胜概率、平局概率，智能推演实战比分，适配小组赛、淘汰赛多种赛事场景。
- 多模型集成加权预测：内置四种机器学习算法，支持用户自定义各模型权重，系统加权融合预测结果，兼顾进攻数据、纪律数据、主场优势、淘汰赛适配性多维度特征。
- 多维数据可视化分析：自动生成球队六维能力对比图、历年世界杯胜率趋势图、模型性能对比图，直观展示两队攻防实力、历史状态、竞技差距。
- 历史交锋数据复盘：智能匹配两支球队历年世界杯交锋记录，展示对战年份、赛事阶段、进球失球、红牌、东道主属性等详细数据。
- AI模型性能评测：集成5折交叉验证、混淆矩阵、准确率、精确率、召回率、F1分数评测，量化各模型泛化能力与稳定性，输出专业算法剖析结论。
- 完整数据统计展示：可视化展示数据集总览、球队场均进球、失球、胜率、红牌纪律、淘汰赛得分、主场胜率等核心数据，自动生成专业预测依据。
- 高颜值动态UI界面：自带渐变背景、漂浮足球、流动草坪、卡片悬浮动效、奖杯闪烁动画，支持自适应布局，视觉效果精致流畅。
3.如何运行或安装：1、项目准备工作
- 将项目源码、数据集文件 世界杯清洗后结构化数据集_含红牌.csv、ronaldo.png、messi.png 放置在同一文件夹目录下，确保文件路径无中文、无空格。
2、安装依赖库
打开终端/CMD，进入项目根目录，执行以下命令批量安装依赖：
pip install pandas numpy scikit-learn altair streamlit
3、启动项目运行
终端进入项目目录，执行运行命令，启动Web可视化平台：
streamlit run 你的文件名.py
示例：
streamlit run worldcup_predict.py
4.依赖项或环境要求：1、基础运行环境
- Python 版本：3.8 及以上（推荐 3.9/3.10 稳定版本）
- 运行系统：Windows / MacOS / Linux 全平台适配
- 核心运行框架：Streamlit
2、必备第三方依赖库
项目运行需安装以下数据分析、机器学习、可视化依赖包：
- pandas：数据集读取、数据清洗与统计分析
- numpy：数值计算、特征向量构建、数据运算
- scikit-learn：机器学习模型训练、预测、交叉验证、模型评估
- altair：交互式数据可视化图表绘制
- base64、math、os：系统内置工具库（无需额外安装）
